Есть [[Основные Цепи, ЦТС-ЦБС]], есть транзакт, что движется по ЦТС, пока всё не закончится.

Команда Queue. Позволяет создать очередь для [[Устройство, SEIZE-RELEASE|Устройства]], либо же другой вещи, подразумевающей критическую секцию.
Команда Depart позволяет убрать очередь.

Второй параметр - количество ввода транзакт за раз

CLEAR - Скорее всего очищает очередь. RESET - полностью обнуляет все данные очереди.

Нулевой вход -- это когда есть мгновенный выход из очереди после входа.
