Это специальная цепь, позволяющая не занимать устройства [[Транзакт|транзактами]]. 
Команда RETURN выплёвывает из цепи задержки в [[Основные Цепи, ЦТС-ЦБС|ЦБС]]
Используется в приоритезации для вытеснений.

Для этой цепи есть команды (блок) PREEMPT и RETURN.
Они добавляют транзакт из любой цепи в эту цепь задержки.

Это позволяет вытеснять транзакты, что уже есть в ЦБС.

PREEMPT[КОД].[PR]\(Приоритет).[МЕТКА] - метка позволит вернуть транзакт в какую-то конкретную точку.

При одинаковых [[Приоритет, PRIORITY]]ах и двух PREEMPT-ах второй просто ничего не сделает.
Если у первого есть приоритет выше - он не вытиснится.
