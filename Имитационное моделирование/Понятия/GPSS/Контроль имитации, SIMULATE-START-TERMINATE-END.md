SIMULATE-END -- Это две команды, которые собирают вообще всё между ними для работы модели. Как фигурные скобочки у классов.

Формат START {Счётчик необходимого количества транзактов},,,{Вывод содерижимого цепей (ставим 1 обычно)}.

TERMINATE {n} - Уменьшает счётчик и удаляет попавший транзакт. Если n не писать, то он будет = 0 и будет просто удалять попавшие в него транзакты.

START'ов может быть несколько. Тогда модель будет срабатывать для каждого отдельно. Это позволяет сделать весьма сложную модель, которая может учитывать результаты предыдущего моделирования.

Например имитация рабочего пространства во время корпоратива, а потом на следующий день - после.
