LINK [ИМЯ],[{формат}] \(FIFO, LIFO, По параметру),{Индикатор связи}.
Индикатор связи - это проверка возможности (по метке) двигать транзакты дальше. Если там стоит занятое устройство - двигаем транзакт в цепь пользователя.

Специфическая, переменная цепь для нужд пользователя.

Извлекать может другой [[Транзакт]] через команду UNLINK.

У UNLINK есть следующие параметры: {Цепь},{Куда выводим},{Сколько},{Определитель края вывода 1},{Определитель края вывода 2} (Последние две штуки не должны спрашивать)

Выводит UNLINK в ЦТС на край вправо.
