#Переписать  -> [Зная случайную функцию распределения величины можно посчитать по обратной функции конкретное значение.] (Это какой-то отдельный метод)

Допустим, есть функция плотности.
Мы знаем площадь под графиком и максимальное значение функции.
Так же есть ограничение функции по X.

Находим $x = {модуль~суммы~границ} * {случайное~число} + {Левая~граница}$.
находим $y={случайное~число} * {максимум~функции}$

Проверяем точку. Успех - попали, промах не попали под функцию.

Позволяет в итоге сгенерировать любое распределение.
Можем дальше по нему генерировать случайные числа.
