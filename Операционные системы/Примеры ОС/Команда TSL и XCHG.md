Test and Set Lock.

Команды сидят в Assembler.

Решение проблемы [[Критическая Секция]]

Использует регистр процессора RX.
Процессор блокирует шину памяти с помощью этой команды, не давая другим процессорам туда доступ.

Альтернатива - XCHG:
Осуществляет атомарный обмен содержимого двух областей памяти. Регистра и слова памяти, например.
