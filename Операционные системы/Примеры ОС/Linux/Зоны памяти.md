Виды:
- ZONE_DMA - страницы для контроллера Direct Memory Access, маленькая (<16МБайт)
- ZONE_DMA32 - То же самое для 32-разрядной версии.
- ZONE_NORMAL - Отображение в адресные пространства обычным образом
- ZONE_HIGHMEM - "Верхняя память" ПК, состоит из страниц которые не могут все и разом отображаться в адресное пространство ядра. (>896 Мбайт)

В структуре zone есть опять целая куча разных параметров, но самые важные там:
- Поле Lock, которое защищает структуру от одновременного доступа разными процессами (Но притом не связанные страницы)
- watermark, которая содержит минимальный, нижний и верхний уровни текущей зоны для оценки расхода памяти в пределах зоны.
- name, для видимости пользователями-разработчиками
