- flags - Состояние страницы. (Изменялось ли? Фиксирована в памяти ли?)
- \_count - счётчик использования в текущий момент времени. Если ушло под 0 -- можно убирать
- virtual - виртуальный адрес страницы
- И ещё пачка других

Эта структура связана со [[Страничная организация памяти|страницами]] именно физической, а не виртуальной памяти.
