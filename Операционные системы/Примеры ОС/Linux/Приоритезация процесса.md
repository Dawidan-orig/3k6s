Используется в [[Планировщик]]
Связь с [[Процесс]]

- Параметр nice $\in (-20;+19)$=0 базово, это и есть приоритет.
Чем меньше параметр - тем большую часть процессорного времени (В [[Квант Времени|квантах]]) получает процесс. 

- Real-time priority $\in [0;99]$, эти приоритетнее nice, они соответствуют стандарту POSIX.
