Связано:[[Процесс]], [[Поток вычисления|поток]], [[Компляторы]]

Компилятор идёт по программе. В какой-то момент выполнения работы процесса P1 появляется P2.

P2 работает с тем же кодом.

У P2 указатель компилцяции УЖЕ на том же месте, что и у P1.
Они оба будут в одном месте, причём второй процесс возьмёт уже сохранённые данные, что были выделены для P1.

У второго указатель на исполняемый файл для Exec() будет пустым, у первого - он будет обычным.

Условно, дальше следующим шагом будет запуск исполняемого файла.

Это будет в [[Лабораторной 2]].

Чтобы не было [[Race condition]] при клонировании процесса (fork(), я так понял):
1) Не копируем ВАП
2) Блокируем родителий до...
	1) Смерти потока
	2) exec() (Запуск исполняемого файла)
