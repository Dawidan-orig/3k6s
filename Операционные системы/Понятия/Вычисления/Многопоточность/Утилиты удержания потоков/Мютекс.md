Это [[Семафор]] с одним параметром (То-есть максимум тут одно устройство)

Объект, который можно блокировать и синхронизировать. Одна штука. Простой, как палка, оттого и быстрый.

Используется для чтения-записи, дают полный доступ для записывающих. Причём записывающий может быть только один, а считывающих - сколько хош.

DECLARE_MUTEX - создание
INIT_MUTEX - создание в runtime
DEFINE_MUTEX(name) -- статическое определение мютекса.

mutex_lock->{...}->mutex_unlock -- позволяет обозначить критическую секцию.
mutex_is_locked -- проверка на захваченность мютекса.

down\_{read-write}\_trylock -- Захват без блокировки, чтение-запись
Перевод из записи в чтение: downgrade_writer().
