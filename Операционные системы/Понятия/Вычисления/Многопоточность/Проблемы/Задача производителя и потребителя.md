Другое название - "задача ограниченного буфера".

У двух процессов есть один ограниченный буфер. Один в буфер помещает, второй - извлекает.

Но если вдруг буфер заполняется, то нужно заблокировать процесс-"производитель". А для этого надо отслеживать освобождённость буфера.

Аналогично и в обратную сторону - если буфер будет пустым.

Для решения задачи используется [[Семафор]] или/и же [[Условные переменные]] (О чём не было сказано, правда.)
