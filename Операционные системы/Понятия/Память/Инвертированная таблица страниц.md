Связь с [[Виртуальная Таблица Страниц]].
Альтернатива для [[Многоуровневые системы страниц]].

Тут всегда два столбца.

Нужна по той причине, что обычная таблица страниц может занимать в ОП много места.
Инверсия тут получается между виртуальным и физическим. Такая таблица указывает из физического адреса - на виртуальный.
И-и-и это конечно приводит к усложнению доступа, ведь приходится пробегать по всей таблице разом и делать сравнения. А не обращаться сразу куда нужно.

Для этого используется TLB: Наиболее интенсивно используемые страницы сохраняются. Там, по факту, только первое обращение долгое (Потому что обращаемся по инвертированной таблице).
Реализация этого TLB может быть через [[Хэш-таблицы]].
